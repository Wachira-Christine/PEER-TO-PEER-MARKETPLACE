*PROJECT DESCRIPTION*

This is a platform where students can offer and request goods and services amongst each other. The system will feature the following:

-Login and registration features with authentication

-Chat system for direct communication

-Browsing and posting of services

-Secure payment integration

It is designed to promote collaboration and resource sharing between students, giving them the chance to earn money from their skills and services.

*STARTER GUIDE*

This guide explains how to run the Student peer-to-peer service marketplace using the following setup:

Frontend: Vite (JavaScript)
Backend: Django (Python)

*Prerequisites*

Before starting, make sure you have the following installed:

-Git

-Node.js and npm

-Django

-Access to the database

*Steps to Run the Project*

Clone the project
Copy the repository to your local machine and navigate into the project folder.

Run the frontend
Navigate to the frontend folder, install all necessary dependencies, and start the development server.
Once the server is running, it usually returns a URL that you can open in your browser to view the frontend.
