PROJECT DESCRIPTION

This is a platform where students can offer and request goods and services amongst each other.The system wil feature the following:
login/registration  features with authentication, chat system ,browsing and posting of services and secure payment integration. It
is designed to promote collaboration and resource sharing between students giving them the chance to earn money from skills and services.
